# LibCurveDollarIncentive
[Git Source](https://github.com/ubiquity/ubiquity-dollar/blob/940040226cc8480b0e7aa65d1592259dfcf013ef/src/dollar/libraries/LibCurveDollarIncentive.sol)

Library adds buy incentive and sell penalty for Curve's Dollar-3CRV MetaPool


## State Variables
### CURVE_DOLLAR_STORAGE_SLOT
Storage slot used to store data for this library


```solidity

```


### _one
One point in `bytes16`


```solidity

```
